# Python Control Flow and Loops

This folder contains a Python notebook demonstrating the use of conditional statements and looping constructs.

The notebook focuses on writing clear and structured logic using `if`, `elif`, `else`, and `for` loops through practical examples.

---

## Topics Covered

- Conditional statements (`if`, `elif`, `else`)
- Logical comparisons
- Looping with `for`
- Basic input validation
- Iterating over ranges, lists, and strings
- Writing readable and structured control flow
