# SQL Fundamentals

This folder contains basic SQL queries written using PostgreSQL.

The queries are organized progressively to demonstrate core SQL concepts including data retrieval, filtering, and joins.

---

## Topics Covered

- Basic SELECT statements
- Filtering using WHERE
- Logical operators
- Aggregations (COUNT, SUM, AVG)
- Joins (INNER JOIN, etc.)

---

## Files

- `01_basic_select_queries.sql` — Basic data retrieval
- `02_filtering_and_conditions.sql` — WHERE, operators, filtering
- `03_joins_and_aggregations.sql` — Joins and aggregate functions

---

## Notes

- Queries are written for learning and practice purposes
- Focus is on clarity and correctness
- PostgreSQL syntax is used
